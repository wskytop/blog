### 运行
```
安装依赖
yarn
运行
yarn dev
打包
yarn build
```
### 目录结构
```
  - dist   // 打包文件
  - public // 发布环境html同级文件目录
  - src    // 项目核心代买
    - compontents // vue组件目录
    - config      // 常量类型配置文件
    - pages       // 页面管理
    - router      // 路由管理
    - service     // api 接口目录
    - static      // 项目静态资源文件
      - utils       // 工具函数
      - styles      // 公共样式管理
    - store       // vux 管理
    - ui          // 自定义的 ui 组件库，通过 ui-xxx 使用
    - App.vue     // 入口 vue 文件
    - main.js     // 入口 js 文件
```
